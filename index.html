GitHub Pages

This page is deployed by the GitHub Pages special configuration for `github.io` websites.

The `website` repo is a React app that gets built with GitHub Actions and automatically pushed to the gh-pages branch of that repo. The GitHub Pages configuration for the `website` repo then makes the result of the build visible on [https://webstartup360.github.io/website](https://webstartup360.github.io/website).

What I haven't done yet is make it so the root ULR actually displays the build result instead of this static page.
